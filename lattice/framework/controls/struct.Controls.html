Struct Controls

source 
pub struct Controls {
    items: Vec<Control>,
    values: ControlValues,
    change_tracker: ChangeTracker,
}
Expand description

A generic abstraction over UI controls that sketches can directly interact with without being coupled to a specific UI framework. See [crate::runtime::gui::draw_controls] for a concrete implementation.
